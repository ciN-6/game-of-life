## addtionnal control

  1. Underpopulation: a user input that tell how many live neibors should be present before the character dies.
  2. Survival: a user input that tells how many Live character with 2 or 3 live neighbors lives.
  3. Overpopulation: a user input that tells how many Live character with > 3 live neighbors dies.
  4. Reproduction: a user input that tells how many Dead character with exactly 3 live neighbors becomes live.
